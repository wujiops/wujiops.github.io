POSIX括号表达式

POSIX方括号表达式是一种特殊的字符类。 POSIX中括号表达式匹配一组字符中的一个字符，就像常规字符类一样。

标准POSIX

正则表达式 匹配的字符范围
[[:alnum:]] 所有字母和数字(包括大写和小写)
[[:alpha:]] 所有字母(包括大写和小写)
[[:blank:]] 所有空白字符，如空格、制表符、换页符等
[[:cntrl:]] 所有控制字符，如退格、回车、换页等
[[:digit:]] 所有数字字符
[[:graph:]] 所有可打印的字符，即非空白、非控制字符的字符
[[:lower:]] 所有小写字母
[[:print:]] 所有可打印的字符，与[[:graph:]]相同
[[:punct:]] 所有标点符号
[[:space:]] 所有空白字符，如空格、制表符、换页符等
[[:upper:]] 所有大写字母
[[:xdigit:]] 所有十六进制数字字符

无标准

正则表达式 匹配的字符范围
[[:ascii:]] 所有 ASCII 字符，即 0-127 范围内的字符
[[:word:]] 所有单词字符，即字母、数字和下划线组成的字符串，不包括空格和标点符号

旧式语法

正则表达式 匹配的字符范围
[[:<:]] Start of Word 所有左尖括号，即 < 字符
[[:>:]] End of Word 所有右尖括号，即 > 字符

随机数

bash默认有一个RANDOM的变量 默认是0~32767

使用 echo $RANDOM 随机产生一个随机数

使用 set|grep RANDOM 查看上一次产生的随机数

[root@localhost ~]# echo $RANDOM
11932

[root@localhost ~]# set |grep RANDOM
RANDOM=28738

举例说明

产生0~1之间的随机数
echo $[$RANDOM%2]

产生0~2之间的随机数
echo $[$RANDOM%3]

...

产生0~100内的随机数
echo $[$RANDOM%101]

产生50-100之内的随机数（原理：0-50的范围+50）
echo $[$RANDOM%51+50]   50-100——>0-50——>%51 +50

产生三位数的随机数（范围是100-999，可以由0-899的范围+100得到）
echo $[$RANDOM%900+100]     100-999 ——>0-899——>%900 +100

使用set|grep RANDOM查看上一次产生的随机数

[root@server ~]# echo $RANDOM
15832
[root@server ~]# set |grep RANDOM
RANDOM=15832
[root@server ~]#
