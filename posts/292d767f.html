关于类的理解

编写有意义的面向对象的代码

类 = 面向对象 ❌

类命名（建议：首字母大写、连接词首字母大写（变量，最好小写、连接词使用下划线连接））、对象（看下文）

实例化（下文）

类只负责去定义或去刻画一些东西，但是它不负责去执行代码。 运行或调用这个类，要在此类的外部。（非要在类内部调用，那这个类就变成了一个函数）

函数和类

# -*- coding: utf-8 -*-
# class定义类，def定义函数
# 函数：
def name(x, y):   # 定义函数名
    sum = x + y   # 定义函数体
    return sum
a = name(1, 2)   # 传入实参
print(a)   # 打印结果

# 类：
class Student():   # 定义类名
    school = '类小学'   # 定义类体，类的特性表现形式，又称数据成员
    sum = '100'
    def __init__(self, school, sum):   # 构造函数，对类初始化。（构造函数__init__，在实例化的时候做一些类的初始化的工作）
        self.school = school   # 定义实例变量，对于对象来说。（类是一个大范围，对象是一个具体事物）
        self.sum = sum    # 如果注释这一行,会去找类中的sum(注一)

    def workbook(self):   # 定义方法（或者称函数），对于类来说。（类方法）
        pass

print(Student.school)    # 类变量没有实例化也可以调用，实例化之后通过实例也可以调用
school1 = Student('abc小学', '1000')   # 实例化，实例化意义：获取一个具体的对象
school2 = Student('哈哈小学', '900')   # （实例化一个对象）

print(school1.sum)   # 打印结果（(解注一)先找实例本身，没有的话就去类里找）

方法 和 函数 的区别：（初学者对此不必纠结。下文只是建议）

在C、C++更多称为函数，在java、C#里更多称为方法（对def name(): \n pass的称谓）

方法是设计层面的一个称谓。函数是程序运行、过程式的称谓

变量在模块里面，称为变量；在类里面，更多称为数据成员。

类 和 对象 有什么区别？

通用定义：类是现实世界或思维世界中的实体在计算机中的反映，它将数据以及这些数据上的操作封装在一起

在Student类里面，学生有很多的特性（名字、年龄、性别等）在类里面是由 数据成员 来表示的，还因该有它们的行为（做作业、考试等）

行为用方法表示。

类设置的好不好关键看行为与特征。

对象：表示一个具体的概念（类比’学生类’，学生类代表学生整体，对象就是具体的一个学生）

换句话说，类只是告诉我们对于一个学生群体来说，每一个学生都会有它的特征名字和年龄等，但是年龄具体取什么值，名字具体取什么值，需要实例化来完成。当类被实例化了以后，就变成了一个对象。

在类被实例化的时候，需要我们往类里面传递具体值，然后才能得到一个具体的对象。

解,注二:

实例方法，对类下面的实例方法，默认会有一个self，调用的时候默认传入，不需要手动添加。

self必须是显示的（和其它语言this.几乎是一样的。其它语言是隐藏的），但是可以是任意字符，只是默认为self（不建议更换成其他字符）

self就是当前调用某一个方法的对象；self代表的是实例，而不是类。（self必须要写，当然你也可以不叫self）

实例方法最大的特点，第一个参数是需要传入一个self，实例方法是和实例对象相关联的，也是实例可以调用的方法

总结：

类包含：变量（类变量、实例变量）；方法（实例方法、类方法、静态方法）；构造函数

方法和变量的关系：从面向对象角度：方法代表类的行为，变量代表刻画一个类的特征。（变量用来刻画名字和年龄，而方法用来描述我们可以做什么事情以及我们自身的一些行为）

在绝大多数情况下面，方法需要对变量做一系列运算，或者说逻辑上的操作，最终去改变变量的状态。当然方法也可以不操作任何变量（少数）

# -*- coding: utf-8 -*-
class Student():   # 类
    sum1 = 0
    name = 'abc'   # 类变量
    age = '0'
    def __init__(self, name, age):   # 构造函数，特别的一个实例方法。
        self.name = name   # 实例变量
        self.age = age
        print(self.name)   # 在实例方法里面操作实例变量,建议加上self
        # print(name)   # 读取的是 形参name ，不加self情况。
        print(Student.sum1)   # 在实例方法里面访问类变量
        print(self.__class__.sum1)   # self里面有一个__class__函数，也可以访问类变量
# 构造函数是一个特殊的实例方法，调用方式不一样。
# 调用构造函数实际上是通过 类(),普通的实例方法调用 对象.do_homework
# 意义不一样，实例方法描述类的行为；构造函数是用来初始化类的特征的。
    def do_homework(self):   # 实例方法(普通的实例方法)
        print('homework')

student1 = Student('小红', 9)   # 实例化，实例对象
print(student1.name)   # 调用
print(Student.name)

静态方法

大部分可以使用类方法，几乎和普通函数没区别。（不推荐使用静态方法，很熟练无所谓。）

# -*- coding: utf-8 -*-
class Student():
    sum = 0
    def __init__(self, name, age):
        self.name = name
        self.age = age
        self.__class__.sum += 1
        print('当前人数：' + str(self.__class__.sum))
    def do_homework(self):
        print('homework')
    @classmethod    # 类方法装饰器（高阶函数讲）;决定类方法和实例方法的不同
    def plus_sum(cls):
        cls.sum += 1
        print(cls.sum)
        # print(self.name)   # 不能访问实例变量；下面的静态方法同样不能访问实例变量
    @staticmethod   # 静态方法装饰器。（对象和类均可调用）
    def add(x, y):
        print(Student.sum)   # 基本上和类方法一样，只是没有显式传入cls
        print('This is a static method')

类的不安全

一个倡议：一个类，它下面的一些数据变量，是类非常重要的特征数据，如果要修改类特征的值，不建议通过直接访问变量的方式，来改变变量的状态

所有对于类下面的变量的更改，建议通过方法来完成（大部分语言所提倡）

不要对变量做外部修改，应该通过方法修改

# -*- coding: utf-8 -*-

继承性

面向对象3大特性：继承性、封装性（抽象）、多态性

类的组成：类的特征（实例变量、类变量）；行为（各种各样的方法）

继承性最基本的定义：避免我们定义重复性的方法和变量（建议一个模块只写一个类，java、C#是这样的）

继承性.py

# -*- coding: utf-8 -*-
# 子类里面调用父类的构造函数
from 继承性 import Human
class Student(Human):
    def __init__(self, school, name, age):
        self.school = school
        Human.__init__(self, name, age)   # 这里必须传入self，用类调用实例方法（普通方法的调用）
    def do_homework(self):
        print('homework')
student1 = Student('哈哈', '小明', 9)
print(student1.name)
print(student1.age)
Student.do_homework('')   # 类调用实例方法,这样self就是一个普通的参数
student1.do_homework()   # 对象调用实例方法
# 在很多语言里面类是不能调用实例方法的，但是python可以。
# 缺点：如果父类改名，子类中的所有父类的方法都要改
# 开闭原则：代码对自身扩展是开放的，对自身的更改是关闭的。
# -*- coding: utf-8 -*-
from 继承性 import Human
class Student(Human):
    def __init__(self, school, name, age):
        self.school = school
        super(Student, self).__init__(name, age)   # 最好使用super关键字调用父类构造函数
    def do_homework(self):
        print('homework')
        super(Student, self).do_homework()   # 注五，调用父类实例方法
student1 = Student('哈哈', '小明', 9)
print(student1.name)
print(student1.age)
student1.do_homework()   # 注释掉注五，优先调用子类的方法
