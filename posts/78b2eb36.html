转义字符: 特殊的字符(表示无法”看见”的字符、与语言本身语法有冲突的字符)

\n   # 换行
\'   # 单引号, 表示与python语法有冲突的字符, 参考: 'let\'s go'
\t   # 横向制表符
\r   # 回车
>>> print('hello \\n world')   # 将hello \n world输出出来
hello \n world

应用: 打印一个路径但是包含\n,\r等相邻的字符, 就变成了特殊字符

>>> print('c:\nortwind\nortwest')
c:
ortwind
ortwest

>>> print('c:\\nortwind\\nortwest')
c:\nortwind\nortwest

>>> print(r'c:\nortwind\nortwest')   # r 表示这个字符串不是一个普通字符串, 而是一个原始字符串。及所见即所得。
c:\nortwind\nortwest

获取”hello world“中w字符, 两种方法

>>> "hello world"[6]
'w'

>>> "hello world"[-4]
'w'

截取字符(切片操作)

>>> "hello world"[0:4]
'hell'

>>> "hello world"[0:5]   # 这里截取前5个字符
'hello'

>>> "hello world"[0:-1]   # 复数, 代表步长, 从前数0个从后数1个, 截取中间的。
'hello wor'   # 左边舍弃0个, 右边舍弃1个。

>>> "hello world"[3:-2]
'lo wo'

截取world, 用两种方式

>>> "hello world"[6:10]   # 成功, 取6-10
'world'

>>> "hello world"[6:]   # 成功, 解释: 右舍0要忽略0, 因为0代表了第一个数
'world'

>>> "hello world"[:-5]   # 测试左舍0, 忽略0, 成功
'hello'

>>> "hello world"[0:-5]
'hello'

>>> "hello world"[-4:]  # 解释, 套路上面的解释, 左舍-4右不舍(右取4至右不舍)
'world'

>>> "hello world"[-4:-2]   # 左舍-4, 右舍2(右取4至右舍2)
'wo'

>>> "hello world"[2:4]   # 与上面拗口的做对比
'll'

>>> "hello world"[-4:]  # 解释, 套路上面的解释, 左舍-4右不舍(右取4至右不舍)

>>> "hello world"[-4:-2]   # 左舍-4, 右舍2(右取4至右舍2)

>>> "hello world"[:-4]   # 左不舍至右舍4
'hello '

>>> "hello world"[4:]   # 左舍4至右不舍
'o world'

>>> "hello world"[-4:]   # 左舍-4 == 右取4(右取4至右不舍)
'world'

>>> "hello world"[:4]   # 右为正数(左不舍至左取4)

'hell'

理解:

[2:4]: 左右为正数, 看右, 从左 数4个字符, 舍弃前2个

[-4:-2]: 左右为负数, 看左, 从右 数4个字符, 舍弃后边的2个

[2:-4]: 正负数, 左舍2右舍4

[;]: 右/左为空, 代表空的方向是不舍弃

实例:

>>> "hello world"[4]
'o'

>>> "hello world"[-4]
'w'
