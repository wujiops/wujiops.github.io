单机版

下载软件包到服务器进行解压

wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-4.0.20.tgz
tar -xvzf mongodb-linux-x86_64-4.0.20.tgz
mv mongodb-linux-x86_64-4.0.20.tgz /home/mongodb

创建配置相关的目录及配置文件

mkdir -p /home/mongodb/conf
vim /home/mongodb/conf/mongodb.conf    #编辑配置文件
bind_ip=192.168.0.128
port=3717
dbpath=/data/db
logpath=/data/log/mongodb.log
logappend=true
fork=true

mkdir -pv /data/{db,log}

启动服务

mongod -f /home/mongodb/conf/mongodb.conf     #启动mongod
netstat -nlutp |grep 3717

创建用户

db.createUser({user: "root", customData: {description: "superuser"}, pwd: "123456", roles: [{role: "userAdminAnyDatabase", db: "admin"}]})

use roxn
db.createUser({user: "roxn_user", pwd: "111111", roles: [{role: "dbOwner", db: "roxn"}]})

# 2
use admin
db.createUser({user: "root", pwd: "123456", roles: [ "root"]})
# 使用官方推荐的方法 调用passwordPrompt() 提示用户输入密码

use admin
db.createUser({user: "myUserAdmin", pwd: passwordPrompt(), roles: [ { role: "userAdminAnyDatabase", db: "admin" }, "readWriteAnyDatabase" ]})

集群配置

在上述查看用户授权中，对root和roxn_user授完权后执行以下操作

在192.168.0.128、192.168.0.129、192.168.0.130三台机器均安装mongodb，别忘了创建db和log目录创建配置文件，内容如下：
bind_ip=192.168.0.128 # 三台机器仅IP不一样，绑定本机IP
port=3717
dbpath=/home/mongodb/db
logpath=/home/mongodb/log/mongodb.log
logappend=true
fork=true
auth=true
replSet=rs # 集群名称创建集群配置会用到
keyFile=/home/mongodb/key_file # 集群主机间验证文件

生成key_file

导入到另外两台机器，key_file保持一致

openssl rand -base64 756 > key_file

配置集群

单机备份

如果单机版直接修改成集群配置，需要备份数据

  1. 备份
    # -d 库名
# -o 指定备份的目录
./mongodump --port=3717 --username=roxn_user -d roxn -o ./
  2. 恢复
    # --host 指定要连接的主机和端口
# -d 库名
./mongorestore --host=192.168.0.128:3717 -u roxn_user --authenticationDatabase -d roxn ./

本机验证

验证普通用户读写

# 使用普通用户登录
[root@mongodb2 mongodb]# ./bin/mongo --host 192.168.0.128:3717 -u roxn_user -p --authenticationDatabase roxn
# 进入roxn数据库
rs:PRIMARY> use roxn
# 创建几条数据，save，insert均可
rs:PRIMARY> db.table_1.save({'Uid':'mail@roxn.com','Al':['test1@roxn.com','test2@roxn.com']})
WriteResult({ "nInserted" : 1 })
rs:PRIMARY> db.table_1.insert({'Uid':'mail@roxn.com','Al':['test1@roxn.com','test2@roxn.com']})
WriteResult({ "nInserted" : 1 })
rs:PRIMARY> db.table_1.insert({'Uid':'mail@roxn.com'})
