枚举

枚举的意义在于它的标签，不在于它的取值

# -*- coding: utf-8 -*-

from enum import Enum
class VIP(Enum):
    YELLOW = 1
    GREEN = 2
    BLACK = 3
    RED = 4
print(VIP.YELLOW)

枚举的类型、枚举的值、枚举的名字

# -*- coding: utf-8 -*-

from enum import Enum
class VIP(Enum):     # 枚举类型VIP
    YELLOW = 1   # 枚举标签 = 枚举值
    GREEN = 2
    BLACK = 3
    RED = 4
    # RED = 5   相同标签
print(VIP.YELLOW)
# VIP.YELLOW = 5    对YELLOW赋值，改变YELLOW
print(VIP.GREEN.value)    # 打印标签的值
print(type(VIP.GREEN.name))   # 这是一个字符串
print(VIP.GREEN.name)   # 标签的名字
print(VIP.GREEN)
print(type(VIP.GREEN))   # 这是enum的类型（枚举类型）
print(VIP['GREEN'])   # 枚举类['枚举名称']，打印具体的枚举的类型VIP.GREEN

枚举之间的比较

# -*- coding: utf-8 -*-

from enum import Enum
class VIP(Enum):
    YELLOW = 1
    GREEN = 2
    BLACK = 3
    RED = 4
class VIP1(Enum):
    YELLOW = 1
    GREEN = 2
    BLACK = 3
    RED = 4
result = VIP.GREEN == VIP.RED     # 枚举之间比较返回bool值
result1 = VIP.GREEN == VIP.GREEN
# result2 = VIP.GREEN > VIP.RED   # 不可以这样比较
result3 = VIP.GREEN == 2   # 等值比较
result4 = VIP.GREEN is VIP.GREEN   # 枚举之间可以进行身份比较
result5 = VIP.GREEN == VIP1.GREEN   # 不同枚举类型不能比较
print(result)

平常使用枚举经常遇到的问题

  1. 在数据库里存储数值表示枚举类型，和存储标签名表示枚举类型。

  2. 建议使用数字表示（简洁，占空间小）

  3. 在代码里面用数字代表枚举类型，不建议。

  4. 建议在代码里面显式的定义一个枚举类，用枚举类下面的每一个枚举类型来对应数据库的每一个数值

例如：

if a==1:   #不建议
    pass
if a==2:
    pass
if a==VIP.YELLOW:   #建议这样
    pass
if a==VIP.BLACK:
    pass

闭包

函数：在python中是对象，在其它语言只是一段可执行的代码

python：一切皆对象

python可以把函数当作一个参数，传递给另一个函数里。

也可以把一个函数，当作另外一个函数的返回结果

a = 1
b = '2'
c = 'def'
def curve_pre():   # 定义curve_pre函数
    def curve():   # 定义curve函数
        print('This is a function')
    return curve   # 在curve_pre函数中调用curve函数
f = curve_pre()    # 调用curve_pre函数（直接写curve_pre是没有内容的，因为函数没有被调用）
f()   # 调用f函数

闭包：函数在定义时候的由外部环境变量所构成的整体，不会受外部函数的影响

闭包 = 函数 + 环境变量

环境变量一定是在定义在函数的外部（这里是在curve外部），也不能是全局变量（不能在curve_pre上面）。

# -*- coding: utf-8 -*-

def curve_pre():
    a = 25
    def curve(x):    # 函数接受参数x
        return a*x*x
    return curve
a = 10
f = curve_pre()
print(f.__closure__)   # 闭包的环境变量，保存在closure里面
print(f.__closure__[0].cell_contents)   # 取出闭包的环境变量
print(f(2))

闭包的意义：保存的是一个环境。

他把函数调用的现场保存起来了

单一的函数是不能决定运行结果的，还需要环境变量

# -*- coding: utf-8 -*-

def curve_pre():
    a = 25
    def curve(x):
        return a*x*x
    return curve
def curve_pre1():   # 不同的闭包
    a = 25
    def curve1(x):
        return a*x*x
    return curve1
# -*- coding: utf-8 -*-

def f1():
    a = 10
    def f2():
        a = 20   # 这里a会被python认为局部变量，不叫作闭包
        # c = 20 * a   # 这里是引用上面的变量a，所以才是闭包。（关键在于这里的a是引用的上面的环境变量）
        print(a)   # 从外往里，这里a=10，f2函数没有被调用
    print(a)   # 这里a=20，是f2函数被调用了
    f2()
    print(a)   # 这里a=10，是调用的f1函数

全局变量编程

不建议，因为不能保证其它函数改变全局变量

# -*- coding: utf-8 -*-

origin = 0
def go(step):
    global origin   # global声明全局变量，表示这是一个全局变量，而不是局部变量
    new_pos = origin + step
    origin = new_pos
    return new_pos
print(go(2))
print(go(3))
print(go(6))

闭包的作用

保护现场功能，记忆了上一次调用的状态

函数式编程

# -*- coding: utf-8 -*-

origin = 0
def factory(pos):
    def go(step):
        nonlocal pos   # 声明变量在整个闭包中都有效
        new_pos = pos + step
        pos = new_pos
        return new_pos
    return go
tourist = factory(origin)
print(tourist(2))
print(origin)   # 这样验证全局变量没有被改变。
print(tourist.__closure__[0].cell_contents)
print(tourist(3))
print(origin)
print(tourist.__closure__[0].cell_contents)
print(tourist(6))
print(origin)
print(tourist.__closure__[0].cell_contents)
