安装

约定

软碟通 版本9.7 软碟通9.7
esxi 版本6.7 VMware vSphere 6.7
下载好esxi6.7需要注册账户下载(免费)，或者通过第三方分享下载。
下载好ultraiso可以免费试用

esxi系统安装

打开软碟通软件，选择镜像，点击启动，写入硬盘映像

写入，等待完成

完成后，写入便捷启动

选择syslinux写入即可

选择Linux服务器安装syslinux

yum install syslinux -y

把menu.c32复制出来

cd /usr/share/syslinux
# 1.使用sftp工具导出
# 2.使用lrzsz导出
yum install lrzsz -y
sz menu.c32

把menu.c32复制到U盘的根目录，选择覆盖

U盘插入服务器，开机F8选择U盘启动

欢迎安装界面，选择Continue

用户协议，F11同意继续

扫描设备，等一会就好

注意！这里一定确认是你的硬盘
这里是安装esxi需要的硬盘，不要选择自己的U盘，除非你想把系统安装到U盘中

输入设备，如果你的键鼠不是特殊的，默认即可

root密码，网页管理台和设置用得到
(最好大小写 符号 数字都有)

选择安装

等待读条

重启

重启完的页面
我的服务器是插入网线，是可以上网的。F2进入设置界面

输入刚才设定的密码，进入设置
输入密码会回到首页，再按一次F2即可

三个网卡设置选项
从上到下分别是，设置网络、重启、测试

回车进入设置，只设置IPv4和DNS即可

IPv4选择手动设置，使用空格键选择，设置完，回车保存

DNS设置和上面一样

使用Esc退出

会提示你保存，保存即可

重启网络

测试网络，给定几个IP，回车测试

全是OK.就行

Esc退回到首页

首页是自己设定的IP和DNS

浏览器输入地址，按红框一步步来

输入用户密码进入管理台

按照红框，选择分配许可证

添加后看过期日期

按照红框，创建虚拟机和VMware一样

补充镜像上载
按照标注序号一步步操作即可，选择存储–>选定其中一块磁盘–>数据存储浏览器–>上载–>选择自己的镜像上传即可.

升级

通过zip包升级

本文以ESXI6.7.0 Update 1为例，将ESXI版本升级至6.7.0 Update 3

1、下载升级包

登陆vmware下载升级包

https://my.vmware.com/zh/group/vmware/patch

2、将下载好的.zip包，上传到存储卷中

3、启用安全Shell（SSH），并关闭所有正在运行的虚拟机

4、进入维护模式准备升级

5、使用xshell连接ESXI，假设ESXI的IP地址192.168.50.127

连接时选择Keyboard Interactive，密码是你当前ESXI的密码

6、升级

输入升级命令进行升级，其中/vmfs/volumes/db1/ESXi670-201912001.zip为升级包所在位置，每个人的路径都不同

esxcli software vib update -d /vmfs/volumes/db1/ESXi670-201912001.zip

Installation Result
Message: The update completed successfully, but the system needs to be rebooted for the changes to be effective.
Reboot Required: true
VIBs Installed: VMW_bootbank_bnxtnet_20.6.101.7-24vmw.670.3.73.14320388, VMW_bootbank_brcmfcoe_11.4.1078.25-14vmw.670.3.73.14320388, VMW_bootbank_elxnet_11.4.1097.0-5vmw.670.3.73.14320388, ……列出的各种升级包。

7、重启

输入命令重启ESXI

reboot

8、重启后可看到版本号改变，升级成功

9、退出维护模式，升级至此完成

ESXi 6.7.0 通过命令行升级

上面写了基于离线包的升级方式。下面写一下在线升级的基本步骤。（实测速度很慢）

1、查看可用的更新包

esxcli software sources profile list -d https://hostupdate.vmware.com/software/VUM/PRODUCTION/main/vmw-depot-index.xml | grep ESXi-6.7

2、执行在线更新（不含TOOLS，防止 No Space Left On Device）

esxcli software profile update -p ESXi-6.7.0-20190802001-no-tools -d https://hostupdate.vmware.com/software/VUM/PRODUCTION/main/vmw-depot-index.xml

3、单独更新TOOLS

esxcli software vib install -v https://hostupdate.vmware.com/software/VUM/PRODUCTION/main/esx/vmw/vib20/tools-light/VMware_locker_tools-light_10.3.10.12406962-14141615.vib

4、重启服务器

reboot
