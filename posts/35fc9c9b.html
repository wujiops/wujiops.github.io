函数参数：

1.必须参数

我们在函数的参数列表里面，所定义的参数，是必须要传递的。

# -*- coding: utf-8 -*-
def add(x, y):   # 这里必须给x，y参数
    # 这里x和y称为：形式参数，形参
    result = x + y
    return result
a = add(1, 2)   # 函数调用的过程中，往函数的参数列表里面传递的参数的实际的取值。--实际参数，a是实参

2.关键字参数

关键字参数和必须参数的区别：在于函数的调用上，不在于函数的定义上。定义是相同的，调用不同。

# -*- coding: utf-8 -*-
def add1(x1, y1):
    result1 = x1 + y1
    return result1
a1 = add1(y1=3, x1=4)   # 关键字参数，在函数调用的时候明确告诉python，给定的实参是具体给哪个形参的

def print_student_files(name, gender='男', age='8',
                        college='哈哈小学'):   # 给形参设置默认值，未设置的形参是一定要传实参的。
    #（不能夹杂着混，比如：（name, gender='男', age）是不可以的。）
    print('我叫' + name)
    print('我今年' + age + '岁')
    print('我是' + gender + '生')
    print('我在' + college + '上学')

print_student_files('小牛')
print('--------分割线--------')
print_student_files('小乐', '女', '7')   # 即使有了默认值，实参传递的值也会覆盖默认值
print('--------分割线--------')
print_student_files('小红', '女', college = '喜喜小学')   # 关键字赋值，是不用考虑传参顺序的。

#（同上，不能夹杂混和，比如（'小红', gender='女', '8'），这样子不行的。反过来关键字参数中间夹杂着实参也不行。）
