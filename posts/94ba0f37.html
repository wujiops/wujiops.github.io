需求：

hexo搭建个人网站，使用OSS+CDN加速。（涉及到域名备案（又涉及到服务器购买）、因为购买了ECS打算利用一下做Git仓库，做web访问）

阿里云备案免费，服务器要钱具体看下图：

买了一个最便宜的服务器102元/年，好像有99元/年的，域名roxn.cn首年29元/年，续费39元/年

备案成本：102+29=131元

以后花费：39+OSS+CDN≈100左右，服务器预算超过300元/年，打算用GitHub pages+免费CDN，年费0元。

接着介绍git仓库搭建情况，及上线

系统情况：

# 系统版本
[root@localhost ~]# cat /etc/os-release 
NAME="OpenCloudOS"
VERSION="8.6"

软件安装

# 均为yum安装
[root@localhost ~]# yum -y install nginx vim git
[root@localhost ~]# systemctl enable nginx
[root@localhost ~]# systemctl start nginx

git仓库

[root@localhost ~]# cd /home/
[root@localhost home]# git init --bare roxn.git
[root@localhost home]# ls
roxn.git

本地仓库连接远程仓库

git remote add origin ssh://root@<服务器IP>:<ssh端口>/home/roxn.git
# hexo _config.yml 配置hexo d命令上传的仓库地址
# repository: ssh://root@<服务器IP>:<ssh端口>/home/roxn.git

SSH免密登录

指明ECS服务器IP地址和ssh端口

$ ssh-copy-id root@<服务器IP>: -p <ssh端口>

服务器WEB自动获取

[root@localhost ]# cd /usr/share/nginx/
[root@localhost nginx]# rm -fr html
[root@localhost nginx]# git clone ssh://root@localhost:222/home/roxn.git html
[root@localhost nginx]# cd html/
[root@localhost html]# ls
404       baidusitemap.xml  categories  cv           index.html  libs    posts    search.xml   tags
archives  baidu_urls.txt    css         favicon.png  js          medias  project  sitemap.xml
[root@localhost html]# cd /home/roxn.git/hooks/
[root@localhost hooks]# cat post-receive   # 每次更新仓库会触发此脚本
#!/bin/bash  
cd /usr/share/nginx/html
unset GIT_DIR
git fetch --all &> /dev/null 
git reset --hard origin/master &> /dev/null   # 到这里会更新html目录。
sh /home/shell/information.sh   # 这里到下面是自定义的脚本，获取两次hexo d之间的commit信息
dingtalk   # 发送钉钉通知
sleep 1 
sh /opt/python-dingtalk/splicing_url.bash   # 整理所有的url，上传到百度收录。

404页面

# 因为html是一个本地git仓库，注意配置访问.git转到404。
    location ^~ /.git {
        return 404;
    }
