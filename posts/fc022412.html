变量

变量的意义和 赋值符号 =, ‘把列表复制给变量A’
命名要有意义, 用简洁的英文单词表示一个变量

>>> A = [1,2,3,4,5]   # 没有意义
>>> print(A)
[1, 2, 3, 4, 5] 
>>> skill = ['新月打击']   # 技能为新月打击, 可以让读者快速的 读懂变量的意义
>>> print(skill)
['新月打击']

变量基本规范

1.变量名首字符不能是数字

2.变量名由 字母, 数字, 下划线 组成

3.系统的 保留关键字 不能在变量名中

['False', 'None', 'True', 'and', 'as', 'assert', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']

4.变量名区分大小写

5.变量本身是没有类型的, 可以是字符串str、元组tuple、列表list、集合set、数字int,float,tuple、布尔数bool、字典dict

>>> type = 1   # type虽然不是保留字, 但是不建议用作变量名, 类似还有print
>>> type(1)

作业: 思考下面的代码: 

>>> a = 1
>>> b = a
>>> a = 3
>>> print(b)
1
>>> a = [1,2,3]
>>> b = a   # 将a列表赋值给b
>>> a[0] = 'a'   # 把a列表的第1个元素替换为'a'
>>> print(a)   # a列表改变
['a', 2, 3]
>>> print(b)   # b也会变, a列表赋值给b列表, a列表中元素改变, b中的也会变
['a', 2, 3]

int值类型 & list引用类型

区别: int是值类型, list是引用类型

int str tuple(不可改变), list set dict(可变)

【复习: int整型 str字符串 tuple元组 list列表 set集合 dict字典 float浮点数 bool布尔数 [36j]complex复数, 序列字符串列表和元组】

>>> a = a + 'python'   # 把'hello' + 'python'赋值给a

tuple和list区别

>>> a = [1,2,3]
>>> id(a)   # id()查看对象的内存地址
2563202131272
>>> hex(id(a))   # hex()十转十六进制, oct()十转八进制, bin()十转二进制 
'0x254cac0a548'
>>> a[0] = '1'
>>> id(a)
2563202131272

tuple: 不可改变, list可以

>>> a = (1,2,3)
>>> a[0] = '1'
Traceback (most recent call last):
  File "<pyshell#32>", line 1, in <module>
    a[0] = '1'
TypeError: 'tuple' object does not support item assignment

list列表可以增加, 元组tuple不可以。

>>> b = [1,2,3]   # list列表可以增加, 元组tuple不可以。
>>> b. append(4)
>>> print(b)
[1, 2, 3, 4]
>>> c = (1,2,3,4)
>>> c. append(4)
Traceback (most recent call last):
  File "<pyshell#37>", line 1, in <module>
    c. append(4)
AttributeError: 'tuple' object has no attribute 'append'

多维序列

二维和三维用的较多

>>> a = (1,2,3,[4,5,6])
>>> type(a)
<class 'tuple'>
>>> a[3][2]
6
>>> type(a[3])
<class 'list'>
>>> a = (1,2,3,[4,5,(6,7)])
>>> a[3][2][1]
7
>>> type(a[3][2])
<class 'tuple'>
>>> a[3][2] = 'a'   # 注意改变的是列表不是元组。[4,5,(6,7)]是列表, 而元组(6,7)是列表的元素, 所以可变, 但是元组中的6, 7元素是不可变的。
>>> print(a)
(1, 2, 3, [4, 5, 'a'])
>>> >>> a = (1,2,3,[4,5,(6,7)])
>>> print(a[3][2][0])
6
>>> a[3][2][0] = 'b'   # 元组中的6, 所以不可变

  File "<pyshell#13>", line 1, in <module>
    a[3][2][0] = 'b'
TypeError: 'tuple' object does not support item assignment
